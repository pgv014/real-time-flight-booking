# Real-Time Flight Booking System

A modern real-time flight booking platform built using Next.js, Supabase, and Tailwind CSS.

## Features

- Flight search
- Real-time seat availability
- Seat booking system
- Passenger information handling
- Booking cancellation
- My Bookings dashboard
- Realtime updates using Supabase
- Responsive UI
- PWA support

## Tech Stack

- Next.js
- React
- Supabase
- Tailwind CSS
- TypeScript

## Setup Instructions

1. Clone the repository

```bash
git clone <repo-url>
```

2. Install dependencies

```bash
npm install
```

3. Create `.env.local`

```env
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
```

4. Run the development server

```bash
npm run dev
```
